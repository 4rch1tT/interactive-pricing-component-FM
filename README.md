## Overview

This project is a responsive interactive pricing component. Users can view different pricing tiers by adjusting a slider and toggle between monthly and yearly billing (with a 25% discount for yearly).

## Features

- Responsive layout for all device sizes
- Interactive slider for page views
- Toggle for monthly/yearly billing
- Custom CSS styling and hover states
